# Perritos Sin Hogar

Sitio web oficial para adopcion y rescate de perros en Chile. La app vive en el directorio `code/`.

## ES - Proposito

- Ayudar a la adopcion responsable y la donacion para rescates.
- Comunicar impacto, mision y formas de colaborar.
- Mantener un sitio rapido, accesible y facil de mantener.

## ES - Estructura

```
code/              # App Next.js (produccion)
vercel.json        # Configuracion de despliegue
```

## ES - Desarrollo

```bash
cd code
pnpm install
pnpm dev
```

## ES - Idiomas

El sitio incluye un selector de idioma via GTranslate (traduccion automatica) para:
- es (predeterminado)
- en
- zh-CN

## ES - Flujo de trabajo (vibecoding)

1. Definir objetivo y metricas (conversion, accesibilidad, SEO).
2. Plan de cambios breve y enfocado.
3. Implementar en bloques pequenos y validables.
4. Revisar impacto visual y de contenido.
5. Documentar decisiones en el changelog o PR.
